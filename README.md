# CRM Clientes

Projeto simples de API REST para gerenciamento de clientes.

## Tecnologias
- Java
- Spring Boot
- MySQL

## Funcionalidades
- Cadastro de clientes
- Listagem de clientes

## Como executar
- Rodar a aplicação pelo IntelliJ ou via Maven
- A API estará disponível em: http://localhost:8080

## Endpoints
- GET /clientes
- POST /clientes
